## Ejercicios

Ejercicio 1
Al pulsar un botón se muestra el mensaje "¡Hola Clase!" en pantalla.

Ejercicio 2
Un botón cambia el color de un texto a rojo.

Ejercicio 3
Se crea un contador que aumenta cada vez que se hace clic en un botón.

Ejercicio 4
Muestra la hora actual del sistema al pulsar un botón.

Ejercicio 5
Permite escribir un nombre en un input y muestra un saludo personalizado.

Ejercicio 6
Suma dos números introducidos por el usuario y muestra el resultado.

Ejercicio 7
Permite ocultar y mostrar un texto usando un botón.

Ejercicio 8
Cambia una imagen por otra al hacer clic en un botón.

Ejercicio 9
Añade elementos a una lista a partir de lo que se escribe en un input.

Ejercicio 10
Simula un semáforo que va cambiando entre rojo, naranja y verde.
